### 字符数组的使用
字符数组的输入
```
const int N = 101;
char str1[N];
cin.getline(str1, N);
```

字符数组的遍历
```c
char *p = str1;
while (*p) {
    cout << *p;
    p++;
}
```

### 使用更加方便的string
string类型的输入
```
string str2;
cin >> str2;
```

string类型的输出
```
for (int i = 0; i < len; i++) {
    cout << ' ' << str[i];
}
```
