如果您是 Windows 的 Linux 子系统（WSL）用户，要安装 Rust，请在终端中运行以下命令，然后遵循屏幕上的指示。

```sh
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

如果报错 linker `cc` not found 则安装 `sudo apt install gcc`

```sh
cargo new greeting
cd ./greeting
cargo build 
cargo run
```
