
## 阿里云加速

单个项目生效，在项目中的 build.gradle 修改内容

```groovy
allprojects {
    repositories {
        maven {
            url 'http://maven.aliyun.com/nexus/content/groups/public/'
        }
    }
}
```

配置全局加速

```groovy
allprojects{
    repositories {
        maven {
            url 'https://repo.huaweicloud.com/repository/maven/'
        }
    }
    buildscript {
        repositories {
            maven {
                url 'https://repo.huaweicloud.com/repository/maven/'
            }
        }
    }
}
```
