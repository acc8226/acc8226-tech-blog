## ac 网址导航

github 为主要平台
gitee 作为备份

推送 github 会自动 mirror 到 gitee。因此只设置 github 为 origin 即可。

```sh
github  git@github.com:acc8226/nav.git
```

项目主页 <https://gitee.com/acc8226/nav>
