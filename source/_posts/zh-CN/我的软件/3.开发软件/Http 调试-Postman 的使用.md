## 问题总结

**在 postman 的 Collection 请求方式为 Bearer Token 报错 异常信息: `Unable to read JSON value: "`**
将录入框中去掉 Bearer 就好了。

**解决 postman 报错：Error: Maximum response size reached**
修改Postman设置：通用-max response size in MB
