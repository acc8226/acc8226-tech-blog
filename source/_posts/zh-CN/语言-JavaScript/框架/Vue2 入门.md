使用 cli 工具。

Vue CLI 现已处于维护模式!

现在官方推荐使用 create-vue 来创建基于 Vite 的新项目。另外请参考 Vue 3 工具链指南 以了解最新的工具推荐。

安装：

```sh
npm install -g @vue/cli
# OR
yarn global add @vue/cli
```

创建一个项目：

```sh
vue create my-project
# OR
vue ui
```
