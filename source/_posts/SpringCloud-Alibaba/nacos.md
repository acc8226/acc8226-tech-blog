单机模式启动

```cd
cd nacos-server-2.1.0\bin
startup.cmd -m standalone
```

## 功能

导出查询结果：导出【所有】内容。

导出选中配置：只导出选中的内容。

报错

```text
Description:

Application failed to connect to Nacos server: "127.0.0.1:8848"

Action:

Please check your Nacos server config
```
